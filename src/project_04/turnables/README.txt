This program uses processing.org to create a canvas where you can draw
using circles, squares, rectangles, and triangles.

Running This Program
    You can run this program from the .jar file,
    TCollins_Project_04.java file (update the classpaths before you do)
    or the ShapePaint.java file.

    It's important to ensure you have the classpaths updated as the
    .java files were from 2 different projects that were interacting
    with each other. It was specifically the project ShapePaint.java
    was a part of was being included into the project
    TCollins_Project_04.java was made in.

KeyBinds:
    1:    Changes which shape will be painted.
    2:    Changes the outline color of the shape.
    3:    Changes the fill color of the shape.
    4:    Changes the color of the background.
    5:    Rotates the rectangle 90Â° at a time.
    -/_:  Decreases the size of the shape.
    +/=:  Increases the size of the shape.

    Left Mouse Button:  Paints the current shape.
    Right Mouse Button: Erases the last shape painted.

    Backspace: Erases the last shape painted.
    DEL:       Clears canvas i.e. erases all the shapes.
    0:         Saves a screenshot of the current canvas.
    Escape:    Exits the program.
