Pokedex Project
A first generation pokedex

Favorite Pokemon: Lucario, Haunter, and Golem

Requirements
- JDK 24+
- images need to be in this directory 
    "src/project_05/src/main/java/data/image/"

How To Use
- Select any Pokemon listed on right panel to display detailed
    information on the left panel.
- Click any of the four sorting buttons to sort them accordingly

Features
- All sorting algorithms sort in ASCENDING order
    - sort by NUMBER sorts by ID
    - sort by NAME sorts by the Pokemon name
    - sort by POWER sorts by the power stat
    - sort by TYPE sorts by the types of the Pokemon
- Left panel shows a sprite of the pokemon and detailed stats
    Health, Speed,
    Attack, Defense,
    Sp. Attack, Sp. Defense
    Height, Weight
    - The catch rate for the pokemon is shown next to its name in
      this panel.

Troubleshooting
